Class Summary
AddressType This domain class defines the different types of mailing address that are associated with a StreetAddress instance.
CommentPersonContact This class extends edu.umn.edu.Comment by tying it to a particular edu.umn.ncs.PersonContact instance.
ContactRole This class assigns a particular contact role between two people.
ContactRoleType This class provides a holder for all allowable contact types, and is used by the edu.umn.ncs.ContactRole domain class.
Country This domain class represents a country somewhere on this planet earth.
DwellingUnit This class represents a single dwelling unit, commonly referred to as a deliverable address.
EmailAddress This class represents a single email address along with all of the required tracking fields associated with the entity.
EnrollmentType This domain class is used to classify the types of study enrollment as used by the edu.umn.ncs.Subject class.
Household This domain class is used to define a single family unit.
PersonContact This domain class is not used directly, it is inherited by other classes.
RelationshipType This domain class defines the different types of relationships that are associated between People instances.
Source This class defines a source of information for a particular data element.
Subject This class represents a subject enrolled in a particular study or substudy.
